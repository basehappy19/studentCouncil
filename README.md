# studentCouncil

This is a web application designed for managing student council activities. It includes a **Client** built with Next.js 15 and a **Server** built with Express.js.

---

## Features

### Client (Next.js 15)
- Responsive design for seamless user experience.
- Dynamic pages for student profiles, event management, and announcements.
- Authentication and authorization.
- API integration with the backend.

### Server (Express.js)
- RESTful APIs for user management, event creation, and council activities.
- Secure user authentication (JWT).
- Database management (e.g., Prisma with mysql).

---

## Installation

### Prerequisites
- Node.js (v22 or later)
- npm or yarn
- mysql or any other database you plan to use
